== Installation ==

Please follow these steps to install the plugin.

1. Upload `wp-sticky-side-buttons` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'WP Sticky Side Buttons' menu in WordPress
1. Place `<?php
if(function_exists('wordpress_Sticky_btns')) {wordpress_Sticky_btns();}?>` in your templates
